# FileNavigation

[सभी भाषाएँ](../../README.md) · [नवीनतम रिलीज़](https://github.com/CSXFanMeng/filenavigation/releases/latest)

FileNavigation Rust + Tauri से बना डेस्कटॉप अनुप्रयोग है जो चुनी गई स्थानीय डायरेक्टरी में फ़ाइलें और फ़ोल्डर खोजता है। खोज पूरी तरह स्थानीय कंप्यूटर पर चलती है; नाम, पथ या सामग्री अपलोड नहीं होती।

## विशेषताएँ

- स्थानीय फ़ाइल और फ़ोल्डर खोज
- रद्द करने और लाइव प्रगति वाला पूर्णतः असिंक्रोनस Rust/Tauri बैकएंड
- परिणाम फ़िल्टर, प्रकार फ़िल्टर, क्रमबद्ध करना और क्रमिक रेंडरिंग
- Lucide आइकन और सुलभ फोकस वाली आधुनिक रिस्पॉन्सिव UI
- ऐप द्वारा बनाई सीमा और स्थानीयकृत नियंत्रणों वाली फ्रेमरहित विंडो
- स्थिर कार्यक्षेत्र और स्वतंत्र रूप से स्क्रॉल होने वाली परिणाम सूची
- अलग भाषा फ़ाइलों में 20-भाषा इंटरफ़ेस
- नवीनतम GitHub Release और वर्तमान भाषा के नोट्स की जाँच
- इंस्टॉलर की अखंडता जाँचने के लिए डाइजेस्ट प्रदर्शन

## डाउनलोड

- Windows: `.exe` और `.msi`
- macOS: `.dmg` और `.app`
- Linux: `.deb`, `.rpm` और `.AppImage`

[GitHub Releases](https://github.com/CSXFanMeng/filenavigation/releases/latest) से डाउनलोड करें।

## विकास और बिल्ड

```bash
npm install
npm run tauri:dev
npm run tauri:build
```

यदि PowerShell `npm.ps1` रोकता है तो `npm.cmd` उपयोग करें। आउटपुट `src-tauri/target/release/bundle/` में मिलता है।

## OpenAI Codex और GPT-5.6 के साथ निर्मित

OpenAI Codex और GPT-5.6 का उपयोग आर्किटेक्चर, रद्द की जा सकने वाली असिंक्रोनस खोज, मॉड्यूलर अंतरराष्ट्रीयकरण, रिस्पॉन्सिव और फ्रेमलेस UI डीबगिंग, क्रॉस-प्लेटफ़ॉर्म CI व रिलीज़ ऑटोमेशन, बहुभाषी दस्तावेज़ और निरंतर बिल्ड व परीक्षण सत्यापन के लिए इंजीनियरिंग सहयोगी के रूप में किया गया।

AI का उपयोग केवल विकास के दौरान होता है। FileNavigation रनटाइम पर OpenAI सेवाओं को कॉल नहीं करता और फ़ाइल नाम, पथ, खोज या फ़ाइल सामग्री किसी मॉडल को नहीं भेजता।
