# FileNavigation

[Все языки](../../README.md) · [Последний выпуск](https://github.com/CSXFanMeng/filenavigation/releases/latest)

FileNavigation — настольное приложение на Rust + Tauri для поиска файлов и папок в выбранном локальном каталоге. Поиск полностью выполняется на компьютере; имена, пути и содержимое файлов не загружаются в сеть.

## Возможности

- Локальный поиск файлов и папок
- Полностью асинхронный Rust/Tauri-бэкенд с отменой и живым прогрессом
- Фильтрация, сортировка и постепенный вывод результатов
- Современный адаптивный интерфейс с иконками Lucide и доступным фокусом
- Окно без системной рамки с границей приложения и локализованными кнопками
- Фиксированная рабочая область и независимо прокручиваемый список результатов
- Интерфейс на 20 языках в отдельных файлах локализации
- Проверка последней GitHub Release с заметками на текущем языке
- Отображение дайджестов установщиков для проверки целостности

## Загрузки

- Windows: `.exe` и `.msi`
- macOS: `.dmg` и `.app`
- Linux: `.deb`, `.rpm` и `.AppImage`

Пакеты доступны в [GitHub Releases](https://github.com/CSXFanMeng/filenavigation/releases/latest).

## Разработка И Сборка

```bash
npm install
npm run tauri:dev
npm run tauri:build
```

Если PowerShell блокирует `npm.ps1`, используйте `npm.cmd`. Результаты находятся в `src-tauri/target/release/bundle/`.

## Языки И Обновления

Переводы находятся в `src/i18n/locales`. [RELEASE_NOTES.md](../../RELEASE_NOTES.md) использует блоки `<!-- lang:xx -->`. Проверка обновлений выбирает текущий язык, затем английский и полный текст Release.
