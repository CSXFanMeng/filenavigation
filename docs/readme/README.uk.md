# FileNavigation

[Усі мови](../../README.md) · [Останній випуск](https://github.com/CSXFanMeng/filenavigation/releases/latest)

FileNavigation — настільний застосунок на Rust + Tauri для пошуку файлів і папок у вибраному локальному каталозі. Пошук повністю виконується на комп’ютері; назви, шляхи та вміст не завантажуються в мережу.

## Можливості

- Локальний пошук файлів і папок
- Повністю асинхронний Rust/Tauri-бекенд зі скасуванням і живим прогресом
- Фільтрування, сортування та поступове відображення результатів
- Сучасний адаптивний інтерфейс із піктограмами Lucide та доступним фокусом
- Вікно без системної рамки з межею застосунку й локалізованими кнопками
- Фіксована робоча область і незалежно прокручуваний список результатів
- Інтерфейс 20 мовами в окремих файлах локалізації
- Вікно оновлень у заголовку з прокручуваними локалізованими нотатками GitHub і підписаним автоматичним встановленням
- Вікно налаштувань зі збереженням світлої й темної тем
- Відображення дайджестів інсталяторів для перевірки цілісності

## Завантаження

- Windows: `.exe` і `.msi`
- macOS: `.dmg` і `.app`
- Linux: `.deb`, `.rpm` і `.AppImage`

Завантажуйте з [GitHub Releases](https://github.com/CSXFanMeng/filenavigation/releases/latest).

## Розробка Та Збірка

```bash
npm install
npm run tauri:dev
npm run tauri:build
```

Якщо PowerShell блокує `npm.ps1`, використовуйте `npm.cmd`. Результати збірки містяться в `src-tauri/target/release/bundle/`.

## Мови Та Оновлення

Переклади містяться в `src/i18n/locales`. [RELEASE_NOTES.md](../../RELEASE_NOTES.md) використовує блоки `<!-- lang:xx -->`. Перевірка оновлень обирає активну мову, потім англійську, а далі повний текст Release.

## Створено З OpenAI Codex І GPT-5.6

OpenAI Codex і GPT-5.6 використовувалися як інженерні помічники для проєктування архітектури, асинхронного пошуку з можливістю скасування, модульної інтернаціоналізації, налагодження адаптивного безрамкового інтерфейсу, автоматизації кросплатформних CI та релізів, багатомовної документації, а також постійної перевірки збірок і тестів.

ШІ використовується лише під час розробки. FileNavigation не звертається до сервісів OpenAI під час роботи та не надсилає моделі назви файлів, шляхи, пошукові запити чи вміст файлів.
